## What It Does

This utility is a focused web calculator for Fortnite's Fight the BRAINROT game mode. By entering your character stats, you can get a fast read on combat power and use that information to guide gear choices and upgrades. The tool handles the in-game variables for you so there is no need to do the math by hand.

Version 1.0 adds mutation support, letting you compare how different mutation settings change the final power total. It also includes reverse calculation, which works from a target power value back to the stats required to reach it. The update is aimed at improving result accuracy while broadening the set of supported setups for both casual and competitive play.
